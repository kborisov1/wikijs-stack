# Wiki.js Docker Stack

This repo contains a full Docker Compose setup for:

- Wiki.js
- PostgreSQL
- Nginx reverse proxy
- Prometheus monitoring
- Grafana dashboards
- cAdvisor and Node Exporter

## How to start

1. Clone the repo:
   git clone https://github.com/kborisov1/wikijs-stack.git
2. Enter the folder:
   cd wikijs-stack
3. Start all services:
   docker compose up -d
